# 🎓 Anderson ENT - Приложение для подготовки к ЕНТ

**anderson-ENT** - это полнофункциональное веб и мобильное приложение для подготовки студентов к ЕНТ (Единое национальное тестирование) по химии, биологии и анатомии.

## 🚀 Функции

✅ **Регистрация и вход** - Безопасная аутентификация с JWT  
✅ **Интерактивные тесты** - Практика с моментальной проверкой  
✅ **Теория и конспекты** - Подробные объяснения каждой темы  
✅ **Видео-уроки** - Интеграция с YouTube  
✅ **Отслеживание прогресса** - Система баллов и статистика  
✅ **Прогнозирование ЕНТ баллов** - Оценка примерного результата на экзамене  
✅ **Мобильная версия** - React Native для Android (планируется)

## 📋 Технологический стек

- **Фронтенд:** React 18, React Router, Axios, CSS3
- **Бэкенд:** Node.js, Express.js, MongoDB
- **Аутентификация:** JWT (JSON Web Token)
- **Безопасность:** bcryptjs для хеширования паролей
- **Мобильная версия:** React Native (планируется)

## 🛠️ Установка

### Требования
- Node.js >= 14
- MongoDB (локально или облако)
- npm или yarn

### Шаги установки

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/sauqwai-a11/anderson-ENT.git
cd anderson-ENT
```

2. **Установите зависимости:**
```bash
npm run install-all
```

3. **Создайте файл `.env` в корневой папке:**
```env
MONGODB_URI=mongodb://localhost:27017/anderson-ent
JWT_SECRET=your_super_secret_key_here
PORT=5000
NODE_ENV=development
REACT_APP_API_URL=http://localhost:5000/api
```

4. **Запустите приложение:**

**В разных терминалах:**
```bash
# Терминал 1 - Сервер
npm run dev

# Терминал 2 - Клиент
npm run client
```

Или запустите оба одновременно:
```bash
npm run dev
```

5. **Откройте браузер:**
```
http://localhost:3000
```

## 🎯 Как это работает

1. **Регистрация** - Создайте аккаунт
2. **Выберите предмет** - Химия, биология или анатомия
3. **Изучите материал** - Конспекты и видео-уроки
4. **Пройдите тесты** - Проверьте свои знания
5. **Смотрите прогресс** - Отслеживайте улучшения
6. **Готовьтесь к ЕНТ** - Достигните целевого балла!

## 📈 Система прогресса

Приложение автоматически рассчитывает примерный балл ЕНТ на основе:
- Результатов пройденных тестов
- Процента правильных ответов
- Частоты занятий

**Цель:** При ежедневных занятиях в течение 3-6 месяцев достичь **140 баллов**!

## 🔐 Безопасность

- Пароли хешируются с помощью bcryptjs
- JWT для защиты API
- CORS настроен для безопасности
- Валидация данных на сервере
